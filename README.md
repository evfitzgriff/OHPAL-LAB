## Quick start (local)

```bash
# 1) Install dependencies
npm install

# 2) Run dev server
npm run dev
```

Open http://localhost:3000

## Deploy to Vercel

1. Create a new GitHub repo and push this folder.
2. Go to https://vercel.com/import and choose "Import Git Repository".
3. Select your repo. Vercel will auto-detect Next.js and Tailwind.
4. Click **Deploy**. That's it.

## Notes
- Update content in `app/page.tsx`.
- Tailwind utilities in `app/globals.css`.
- No server code required; purely static pages.
